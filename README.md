# Windows-Server-2022-Home-Lab-Project
## Objective
Set up a Windows Server environment to practice Active Directory management and network configuration.
## Tools and Technologies
- VirtualBox
- Windows Server 2022
- Active Directory Tools
- Group Policy Management Console
- Windows 11 Enterprise
## Tasks
-	Installed Windows Server 2022 on a virtual machine and configured it to be a domain controller.
-	Attached Windows 11 virtual machine to the domain controller.
-	Set up and managed Active Directory, including creating and managing user accounts, groups and organizational units.
-	Implemented Group Policy Objects (GPOs) to enforce security policies.
-	Used PowerShell to write scripts to perform administrative tasks and automate them.
-	Reset Active Directory user account passwords.
